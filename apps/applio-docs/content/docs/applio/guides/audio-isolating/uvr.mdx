<Tabs items={['RoFormer', 'MDX-Net', 'Vr Arch', 'Demucs']}>
<Tab value="RoFormer">
  **RoFormer** utilizes a Band-Split module to convert the input complex spectrogram into representations at the subband level *(currently the best)*. 
  
  The RoFormer method contains two pretrained models, **BS-RoFormer** and **Mel-Band RoFormer**, which contain similar but not same structures, currently the best versions of these models are:

 - **BS-Roformer-Viperx-1297:** The best to separate vocals and instrumental.
 - **Mel-Roformer-Viperx-1143:** Separate vocals and instrumental.

  **For higher quality, use custom models**
<Accordions>
<Accordion title="Custom Models">

- **[anvuew Mel Roformer dereverb](https://huggingface.co/anvuew/dereverb_mel_band_roformer/tree/main):** The best model for removing reverb from audios *(only removes stereo reverb)*.

For information on how to use these models in the UVR, go to [Custom Models Information](#custom-models)

<Tab value="MDX-Net">
  **MDX-net** is a prize-winning model framework from an ML enthusiasts contest, designed for audio source separation and enabling improvements from a base model.

  - **MDX23C-InstVoc HQ:** A set of different models that works to deliver a single result.
  - **UVR-MDX-NET Voc FT:** Model focused mainly on separating vocals.
  - **UVR-MDX-NET Crowd HQ 1:** separate background vocals from a main one.
</Tab>
<Tab value="Vr Arch">
**Vr-Arch** is a location-guided audio-visual spatial audio separator model, capable of giving great results in most tasks.

  - **UVR-DeEcho-DeReverb:** Separate Reverb and echo.
  - **UVR-BVE-4B_SN-44100-1:** Multiple background voices separator.
  - **5_Hp-Karaoke-UVR:** Separate Background vocals (best one).
  - **UVR-DeNoise:** Remove Noise and some artifacts.
</Tab>
<Tab value="Demucs">
  **Demucs** is an all-in-one model capable of separating most objects from an audio between 4 to 6 stems. Being their best version of DemucsV4. 
  - **htdemucs:** Default Model, normal proccesing time and good results.
  - **htdemucs_ft:** Slow processing but better results.
  - **htdemucs_6s:** Capability of separing Guitar and Piano stems.
</Tab>

<Step>
### Make the Audio conversion

When the models have been downloaded, you can go back to the home screen to make your separation.

- Select the desired structure and the downloaded model.
- Select your audio, it can be easily downloaded from sites such as [Cobalt.Tools](https://cobalt.tools/)
- Select the output path for the results and click convert.

<Accordions>
<Accordion title="Other Options">
- **Segment Size:** Larger consumes more resources, but may give better results.
- **Overlap:** Amount of overlap between prediction windows.
- **Window Size:** Change the audio processing bucket, a lower number indicates a much slower processing but a better worked result, and vice versa with a higher number.
- **Aggression Settings:** Change the aggressiveness with which the separation will be performed, a higher number can eat important frequencies of the audio.
</Accordion>
</Accordions>

*(This small application applies to any alternative that runs the GUI version of the program.)*

## Alternative 2: MVSEP

MVSEP is a website belonging to the **UVR team** which offers a fast and direct way to use the software in the cloud with a free version and paid upgrades.

To use mvsep, it is as easy as enter to their website (https://mvsep.com/) and registering. Apart from the mentioned models, MVSEP has some exclusive models, which are:

- **BSRoformer (ver 2024.08)** or **MelBand Roformer (ver 2024.08):** The best models to separate instrumental and vocals.
- **Bandit v2:** Remove SFX, speech, music, effects.
- **DeNoise by aufr23 (Standard & Aggressive):** Remove noise and some artifacts.
