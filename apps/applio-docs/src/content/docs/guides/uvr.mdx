**Ultimate Vocal Remover (UVR)** is a powerful, free, and open-source tool for separating audio sources. It's an essential tool for creating the clean, isolated vocal tracks needed for high-quality voice model training.

## How to Install UVR

To use UVR, you'll need to install it on your local machine.

<Steps>
1.  Download the installer from the [UVR website](https://ultimatevocalremover.com/).
2.  Run the installer to set up UVR on your computer.
3.  Once installed, launch UVR and click the **wrench icon** (ðŸ”§) to open the settings.
4.  Go to the **Download Center** and download the separation models you want to use.
</Steps>

<Aside type="note">
  When you run the installer, Windows may flag it as potential malware. This is a false positive due to the installer not being signed with a commercial license. You can review the source code on their GitHub repository if you have any concerns.
</Aside>

## How to Use UVR

<Steps>
1.  **Select Your Audio File:** Choose the audio file you want to process.
2.  **Choose an Output Path:** Select a folder where the separated audio files will be saved.
3.  **Select a Model:** Choose the separation model you want to use. Different models are optimized for different tasks.
4.  **Click Convert:** Click the **Convert** button to start the separation process.
</Steps>
